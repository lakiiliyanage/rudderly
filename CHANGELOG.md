# Changelog

All notable changes to AgentForge are documented here.

---

## Week 9 (May 2026)

- **Conversation persistence** — chat history saved to Supabase; sidebar lists all prior conversations; messages reload on page visit
- **Context window management** — sliding window with summary keeps long conversations within token limits
- **Shareable agent URLs** — auto-generated slugs, public share pages with OG metadata
- **Clone agent** — logged-in visitors can clone any public agent to their own account with one click
- **View counter** — public agent pages track and display total conversations started
